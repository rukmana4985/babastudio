Apa Kegunaan HTML ?
Dengan Menggunakan HTML anda dapat menampilkan informasi diwebsite dalam bentuk :

" Sebenarnya untuk menampilkan informasi pada website tanpa menggunakan HTML pun tetap bisa tetapi anda hanya dapat menampilkan "TEXT" tidak dapat menampilkan gambar , video dan suara . "

Contoh code HTML yang mendasar adalah sebagai berikut :

Code HTML , terdiri dari <> yang disebut tag atau element, dimana tag atau element merepresentasikan tampilan website.

Arti dari <strong></strong> unutk membuat text menjadi tebal .

Jadi untuk membuat tampilan hurup menjadi tebal saja, tidak semudah seperti dalam dokumen WORD di mana cukup dengan fungsi BOLD Text anda langsung menjadi tebal

Apa Kegunaan CSS ?

Untuk mempercantik dan memperindah tampilan website . membuat Website hanya menggunakan HTML, berarti baru membuat tampilan dasar Website , baru fondasi dari tampilan , oleh sebab itu dibutuhkan CSS.

Silahkan Perhatikan Gambar perbedaan tampilan yang menggunakan CSS dan tampilan yang tidak menggunakan CSS.
